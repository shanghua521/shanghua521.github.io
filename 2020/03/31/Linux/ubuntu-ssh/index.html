使用 SSH 连接 Ubuntu

发表于2020-03-30|更新于2021-06-20
|字数总计:479|阅读时长:2分钟|阅读量:

软件准备

FinalShell 软件官网 http://www.hostbuf.com/

因为我们之前安装系统的时候已经安装过 SSH 了，这里我们直接连接就好了

第一步

查看 IP 地址，在命令行输入 ifconfig

这里我的 IP 地址就是 192.168.1.103

打开 FinalShell 并新建一个连接

点击确定，这样我们就连接成功了

这里我们就获得了一个 shell 就可以直接操作虚拟机里的系统了

设置 ROOT 用户

先输入当前用户密码，然后再输入两次 ROOT 账户密码，这样就设置成功了

切换到 ROOT 用户

我们输入 su 命令，然后再输入我们刚刚设置的 ROOT 密码就可以切换到 ROOT 用户了，我们可以从命令行看到，用户从 shanghua 变成了 root

配置 SSH 允许 ROOT 用户远程连接

修改 SSH 配置文件，将 PermitRootLogin 后面的 prohibit-password 修改为 yes

重启 SSH 服务

重启后 ssh 配置文件才会生效哦

编辑 FinalShell 连接改为 root 用户

记得改密码哦，修改完成点击确定，然后打开这个连接

这样就会发现连接上后就是 root 用户了
