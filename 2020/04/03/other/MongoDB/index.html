MongoDB 的安装和使用

发表于2020-04-03|更新于2021-06-20
|字数总计:758|阅读时长:3分钟|阅读量:

安装 MongoDB

  • 进入到 https://www.mongodb.com/download-center/community 即可选择符合你操作系统的安装包
  • 按照安装包指示的流程安装即可
  • 安装完成后将安装后的 bin 目录加入到系统 path 环境变量中

使用 MongoDB

  • 创建目录
    • 打开命令提示符
    • 创建存放数据的目录 md d:\data
    • 启动服务 mongod ‐‐dbpath=d:\data
    • 连接 mongo 192.168.184.134
  • 常用命令
    • 创建数据库 use 数据库名 数据库不存在则会自动创建
      • use spit
    • 插入数据
      • db.数据库名.insert(数据);
      • db.spit.insert({content:"aaa",userid:"1011",nickname:"小雅",visits:NumberInt(902)});
    • 查询数据
      • db.集合名称.find();
      • db.spit.find();
      • 可以发现，每个数据库文档都会自动创建一个 _id 字段，相当于数据库的主键，我们可以插入支持的类型值替换
        • db.spit.insert({_id:"1",content:"aaa",userid:"1012",nickname:"小明",visits:NumberInt(2020)});
    • 条件查询
      • db.spit.find({userid:'1011'})
      • 查询一个 db.spit.findOne({userid:'1013'})
      • 指定条数 db.spit.find().limit(3)
    • 修改数据
      • db.集合名称.update(条件,修改后的数据)
      • 修改 _id 为 1 的记录 db.spit.update({_id:"1"},{visits:NumberInt(1000)})
        • 修改后除了 visits 字段其他字段都不见了，我们可以使用 $set 解决
        • db.spit.update({_id:"2"},{$set:{visits:NumberInt(2000)}})
    • 删除数据
      • db.集合名称.remove(条件) 入过没条件则全部删除 慎用！
      • db.集合名称.remove({visits:1000}) 删除 visits 为 100 的值
    • 统计条数
      • db.spit.count()
      • db.spit.count({userid:"1012"}) 统计 userid 为 1012 的数量\
    • 模糊查询
      • /模糊查询字符串/
      • db.spit.find({content:/流量/})
      • db.spit.find({content:/^加班/})
    • 大于 小于 不等于
      • db.集合名称.find({ "field" : { $gt: value }}) //  大于: field > value
      • db.集合名称.find({ "field" : { $lt: value }}) //  小于: field < value
      • db.集合名称.find({ "field" : { $gte: value }}) //  大于等于: field >= value
      • db.集合名称.find({ "field" : { $lte: value }}) //  小于等于: field <= value
      • db.集合名称.find({ "field" : { $ne: value }}) //  不等于: field != value
    • 包含与不包含
      • db.spit.find({userid:{$in:["1013","1014"]}}) 查询 useid 字段包含 1013 和 1014 的文档
      • db.spit.find({userid:{$nin:["1013","1014"]}}) 查询 useid 字段不包含 1013 和 1014 的文档
    • 条件连接
      • $and:[ { },{ },{ } ] 相当于 SQL 中的 AND
      • db.spit.find({$and:[ {visits:{$gte:1000}} ,{visits:{$lt:2000} }]}) 查询 visits 大于 1000 并且小于 2000 的文档
      • $or:[ { },{ },{ } ] 相当于 SQL 中的 OR
      • db.spit.find({$or:[ {userid:{$gte:"1013"}} ,{visits:{$lt:2000} }]}) 查询 userid 等于 1013 并且 visits 小于 2000 的文档
    • 列值增长
      • db.spit.update({_id:"2"},{$inc:{visits:NumberInt(1)}})
