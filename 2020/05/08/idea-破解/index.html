IDEA 破解教程

发表于2020-05-08|更新于2021-06-19
|阅读量:
  1. 下载破解 jar 包 下载地址
  2. 下载后解压
  3. 打开 IDEA 在试用模式下点击 Help –> Edit Custom VM Option 第一次点击会让创建一个文件，点击确定
  4. 在最后一行加入 -javaagent:你解压后jar包的位置
  6. 重启 IDEA
  7. IDEA 启动后点击 Help –> Register –> License server
  8. 在 Server address 内输入 http://fls.jetbrains-agent.com 点击 ACTIVATE 成功后点击 CONTINUE
  9. DONE

使用方法来自 https://zhile.io/2018/08/25/jetbrains-license-server-crack.html
